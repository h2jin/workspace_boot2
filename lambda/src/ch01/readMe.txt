
람다식 (Lambda expression) --> 람다 표현식

함수형 프로그래밍과 람다식

자바 : 객체지향 프로그래밍
기능을 수행하기 위해서 객체를 만들고 멤버변수, 메서드를 수행하게 코드를 설계한다.

함수형 프로그래밍 방식 (람다식)
: 함수의 구현과 호출만으로 프로그래밍 방식을 지원하는 방식.--> 람다 표현식

함수형 프로그래밍 (FP Functional Programming)

함수형 프로그래밍은 순수함수(Pure Function)을 구현하고 호출함으로써 외부 자료에 
부수적인 영향을 주지 않도록 구현하는 방식이다.

즉, 매개변수만을 사용하여 만드는 함수를 순수함수라고 한다.

배우는 목적--> 실제로 많이 사용하는 표현식이기 떄문에 배워야 함.

람다식 문법 

익명함수 부터 시작하여 --> 람다식 으로 점점 발전

람다식으로 사용할 인터페이스를 만드는 경우 @FunctionalInterface 를 해주는 것이 좋다! 잊지 않고 해줄 것!
